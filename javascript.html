What is JavaScript?

JavaScript, or JS is the programming language used to create interactive functionality in a web document. JavaScript is a scripting language. Instead of a static website, JavaScript allows you to create interactive maps, timely content updates, and more!

What Can You Do With JavaScript?

With JavaScript, you can do some cool things. You can make your website much more interesting by making it no longer static.

Why is JavaScript Important?

A lot of the time, JavaScript does not really change too much about a website, but when it is there it is noticable. The main reason of the importance of JavaScript is that it makes responsive web design much easier for developers.

How Can We Even Use JavaScript With Our Website?

Just like CSS, there are actually multiple ways. We will cover the two most practical ones now.

Method One: External JS:

This is how to use JavaScript in an external file. First, you will need to create another file and give it the .js file extension (for example, "script.js"). This will be the file in which you write all of your JavaScript code. Next, you will have to go to your HTML file and go to the body tag. Unlike CSS, you link JavaScript in the body. Once inside the tag, you will need to insert this code:
<script src="file.js"></script>. Make sure to change the src to the name of your JavaScript file. Now, your file is connected to the HTML and you are free to start coding!

Method Two: Internal JS:

This is how to use JavaScript directly inside the HTML file. In HTML, we have a special tag called the script tag. To use the script tag, simply go to the body of your HTML file, and insert <script></script>. Inside the script tag, you can write all your code and it will be applied to that document. Easy, right?
