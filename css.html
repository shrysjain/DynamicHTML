CSS

Welcome to the CSS section. Here you will learn about CSS, what it does, how it is important, and the basics of connecting and using CSS. We will also use CSS on our document example from the HTML page to make it look better. Let's get started!

What is CSS?

CSS is the programming language used to specify the style of an HTML document. This ranges from things like fonts, to layouts, to colors, and much more. CSS is directly linked to the HTML document. In case you were wondering why the HTML examples looked so basic and ugly, but websites you have seen looked nice, it is likely because of CSS.

What Can You Do With CSS?

You can do a variety of things with CSS. CSS can style almost every HTML tag that creates a visible element on the page. CSS allows you to customize the look and feel of the website to use whatever fonts, colors, design, or almost anything else to your liking.

Why is CSS Important?

CSS is very important when it comes to web development. For most people, everytime you see a website it has some sort of styling to make it look good. Even this website has CSS to make it look good. This very website uses CSS for many things, from the background color, to the fonts, to the font color, sizing, layouts, animations, positioning, and more! To show how much CSS impacts a website, here is this exact webpage without any CSS or styling. Shocking, right? Styling really changes the feel of a website and can be the difference between a user liking the site and disliking it.

How Can We Even Use CSS With Our Website?

There are actually multiple ways to use CSS with an HTML web page. We will cover the two most practical ones now.

Method One: External CSS:

This is how to use CSS in an external file. First, you will need to create another file and give it the .css file extension (for example, "index.css"). This will be the file in which you write all of your CSS code. Next, you will have to go to your HTML file and go to the head tag. Remember how we said the head tag stores important information that isn't displayed? One example is linking to a CSS file. Once inside the tag, you will need to insert this code:
<link rel="stylesheet" href="file.css">. Make sure to change the href to the name of your CSS file. Now, your CSS file is connected to the HTML and you are free to start coding CSS!

Method Two: Internal CSS:

This is how to use CSS directly inside the HTML file. In HTML, we have a special tag called the style tag. To use the style tag, simply go to the head of your HTML file, and insert <style></style>. Inside the style tag, you can write all your CSS code and it will be applied to that document. Easy, right?

Now, we will learn the basic CSS syntax.

How CSS Works

Now that you know all about CSS, let us explain the syntax and format of a CSS file. In CSS, you will use elements in class names to style them. This is the format:
element {
property: value;
}
where element is the name of the element you are styling, property is what you are styling, and value is what you are styling it to. It might seem a little bit complicated at first, but it gets easy.

That is a basic demonstration of CSS. Now, let us break it down.

    * { - This is where we specify the element we are styling. We used the asteriks beacuse that means everything, so this will style everything on the document.

    background-color: red; - This is one example of a CSS property, background-color. If you haven't guessed it, this property sets the background color of whatever element it is nested in. Since we put it in the asteriks, this will make the background color of the entire page red. This property also supports hex, rgb, rgba, and hsl values. Note that every line defining a property and value MUST end with a semicolon at the end.

    font-family: sans-serif; - This is the font family property. Since it applies to everything, this will make all text on the page use the font sans-serif. This can also support custom fonts.

    } - This closes the properties being changed for everything on the website.

    p { - This tells us that now instead of changing everything we will only style the paragraph tag.

    font-size: 20px; - This makes it so that all paragraphs have a font size of 20 pixels.

    font-size: 20px; - Close out.
