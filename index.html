Homepage

Welcome to the homepage of my website. In this website you will learn about HTML, CSS, JavaScript, what they do, and how to use them.

HTML

First we will talk about HTML. HTML stands for HyperText Markup Language. HTML code in a webpage defines the meaning and structure of the content. It contains the elements of the webpage.

CSS

Next is CSS. CSS stands for Casading Style Sheets. CSS is the programming language used to style an HTML webpage. This includes, colors, fonts, placement, backgrounds, and way more. CSS directly links to the HTML file.

JavaScript

Finally we will talk about JavaScript. JavaScript doesn't stand for anything else. JavaScript is used to make webpages interactive. It can interact with HTML elements, as well as the CSS to change the website when the user interacts with it.

So, if you are ready to learn about any or all of these, navigate to the next page with the navagation bar at the top.
