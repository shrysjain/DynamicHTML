HTML

Welcome to the HTML section. Here you will learn about HTML, what it does, how it is important, and also the basics of using HTML. Let's get started!

What is HTML?

HTML is the standard markup language to display documents in a web browser. Typically, web browsers will recieve encoded HTML content and then be able to decode and display the content. Fun fact: this entire website is written with HTML and Sass (Sass is like an extension of CSS which compiles directly into CSS. We will talk about it later).

What Can You Do With HTML?

The answer is lots of things. You can use HTML to make web page, web documents, internet navigation, responsive renders, client or server-side storage, data entry, and much more! The most common thing HTML is used for is for created web pages/documents.

Why is HTML Important?

HTML is very important because the programming language is the modern way to create websites, and HTML is probably the best way we have to describe the structure of information on a webpage. Here is one reason that HTML is the superior programming language for creating websites: Compared to the alternatives, HTML content is much easier to read and access with web crawlers in the browser, therefore reducing load time and improving performance, making HTML a popular favorite among web developers.

This is somewhat basic HTML code. Let's break it down. One more thing: HTML is written in tags. Tags are enclosed with < and > symbols. Some tags, like the paragraph tag, require you to put the content for the tag in between the open and closing braces, and some like the linebreak tag (which we will learn about later) just require opening it and not closing it. Now you are ready to understand the HTML code.

    <!DOCTYPE html> - This line of code is the document type declaration. It is the first line of code required in HTML and XHTML documents. It tells the browser interpreting the code what version of HTML the document is written in.

    <head></head> - This is the head tag of an HTML or XHTML document. This tag contains the metadata of the document. This is important information such as the stylesheet, title, or favicon. Some tags that can be used in the head are the title tag, the meta tag, the style tag, and the link tag, but there are more.

    <title></title> - This is the title tag. This tag is typically placed inside the head tag. The content inside this tag will be displayed above the webpage. On this webpage that you are reading right now, the title is "HTML", and you can see it in your browser above the address bar.

    <body></body> - This is the body tag. It is typically placed outside of but right after the head tag. Unlike the head tag containing information, this contains the actual content of the document. It contains everything, such as the headings, paragraphs, images, hyperlinks, tables, lists, e.t.c.

    <p></p> - This is the paragraph tag. It is typically placed inside the body tag. The content enclosed in this tag is displayed a usually small peice of content on the webpage. This piece of text you are currently reading is inclosed in a paragraph tag.

Please note that these are by no means every tag. This is just a basic layout for a basic HTML webpage. Now, we will use some other tags to create another basic document.
